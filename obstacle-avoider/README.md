# Obstacle Avoider
![Robot Image](robot.jpg)
## Description\
Simple bot which avoids obstacles using ultrasonic sensor.
## Components
- Arduino Uno
- Arduino Motor Shield
- 3 Ultrasonic Sensor
- 2 DC Motors
- Chasis
- Wheels
- 9V Battery
- switch on/off
